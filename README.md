# Coffee Order System (Python)

A simple command-line coffee ordering system built with Python.

This project allows users to:
- View a coffee menu
- Add coffee items to an order
- View the current order
- Checkout and confirm the order

The goal of this project is to practice Python basics such as:
- Classes and objects
- Lists
- Functions
- User input handling

## Features
- Coffee menu with prices
- Add items to an order
- View total price
- Simple checkout process

## How to Run
1. Make sure Python is installed on your machine
2. Clone this repository
3. Run the program:
python main.py
